### CRT Chain (After v6)
```

            â”€â”€â–ş .sys_proc_prx_param (magic 0x1B434CEC, PRX type 0x04)
```

### Key Design Decisions
- **No `-nostartfiles`**: The stock `lv2-sprx.o` provides `_start` via the default linker script. Using `-nostartfiles` would suppress its inclusion.
- **PRX type (0x04)**: Generated by `SYS_MODULE_INFO` macro, which emits the correct PRX type for Cobra VSH compatibility. This replaces the problematic `.sys_proc_param` EBOOT marker.
- **`<sys/prx.h>` already included**: Line 26 of `main.c` had `#include <sys/prx.h>` â€” no new include was needed.
